## üöÄ Quick Start

### Option 1: Direct Deployment to Netlify

1. Save the `index.html` file to your computer
2. Go to [netlify.com](https://www.netlify.com) and sign up/login
3. Drag and drop the `index.html` file into Netlify's deploy zone
4. Your app will be live instantly with a free URL!

### Option 2: GitHub + Netlify

1. Create a new GitHub repository
2. Upload the `index.html` file to your repository
3. Go to Netlify and click "New site from Git"
4. Connect your GitHub repository
5. Deploy with default settings

### Option 3: Local Development

1. Download the `index.html` file
2. Open it directly in any modern web browser
3. No server or build process required!

## üìñ How to Use

### Setting Up a Match

1. Navigate to the **Match Setup** tab
2. Enter home and away team names
3. Add player names (comma-separated) for each team
4. Set match venue and date (optional)
5. Choose match duration
6. Click **Start Match**

### During the Match

1. Use the **Live Match** tab to record events:
   - Click **Home Goal** or **Away Goal** to record goals
   - Select the scorer and optional assist provider
   - Use **Yellow Card** or **Red Card** buttons for disciplinary actions
   - Click **Substitution** to log player changes
   - Use **Start/Pause** to control the match timer

2. All events appear in the **Match Timeline** in real-time

### Viewing Statistics

- **Statistics Tab** - View aggregated team statistics
- **Players Tab** - See individual player performance metrics
